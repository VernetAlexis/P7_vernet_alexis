Avant de pouvoir lancer l'application : 

 - Pour la base de données :
	- Lancer MySQL Workbench
	- Sélectionnez "Local instance MySQL80" (root, localhost:3306)
	- Data Import/Restore
	- Import from Self-Contained File
	- Sélectionnez le fichier "groupomania.sql" fournis
	- Start Import

 - dans le repertoir fontend :
	- npm install
	- npm start

 - dans le repertoir backend : 
	- npm install
	- npm start
